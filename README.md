Учебный веб-проект (Bootstrap + Python)

## Описание проекта

В рамках домашнего задания был разработан простой веб-проект, состоящий из:

- HTML-страниц, сверстанных с использованием Bootstrap
- простого веб-сервера на Python

Проект демонстрирует базовые навыки:
- верстки интерфейса
- работы с Git и GitHub
- создания простого HTTP-сервера

## Верстка страниц

Были реализованы следующие страницы:

- **Главная** — отображает карточки товаров
- **Каталог** — список категорий
- **Категория** — список товаров
- **Контакты** — форма обратной связи

Все страницы:
- сверстаны с использованием **Bootstrap**
- адаптированы под структуру макета
- имеют боковое меню навигации

Bootstrap подключен через CDN:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/css/bootstrap.min.css" rel="stylesheet">
```

Веб-приложение

Реализован простой HTTP-сервер на Python.

🔹 Обработка GET-запроса

При любом GET-запросе сервер возвращает страницу Контакты:

```python
def do_GET(self):
    self._send_html_file("templates/contacts.html")
```

HTML-файл считывается через:

```python
with open(file_path, "r", encoding="utf-8") as file:
```

Тип ответа:

Content-type: text/html
